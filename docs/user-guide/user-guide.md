# User guide - materials

1. Start the app from `src/BolshayaPachkaMaterials.App`.
2. Use the top search field to search by material name and description.
3. Use sorting to order by name, stock, or cost.
4. Use filtering to select one material type; `Все типы` shows all records.
5. The bottom counter shows how many records are displayed from the filtered total.
6. Use page buttons to move through the list; each page shows 15 records.
7. Low stock materials are highlighted red; high stock materials are highlighted yellow.
8. Select records and use the mass update panel to change minimum count.
9. Add/Edit/Delete buttons demonstrate record maintenance and dependency rules.
10. Use `Сбросить данные` to restore initial imported demo data before demonstration.
11. Missing images use `resources/picture.png`.
12. SQL Server scripts are in `database/sqlserver`; run `setup-db.ps1` before demonstrating database deployment.
